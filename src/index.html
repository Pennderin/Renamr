Format Editor

Customize naming expressions

Movie Format

Metadata: {title} {year} {rating}
Video: {resolution} {source} {videoCodec} {hdr} {edition}
Audio: {audioCodec} {channels}
Other: {group}  |  Use / to create folders

Preview:

TV Show Format

Metadata: {series} {season} {episode} {title} {year}
Video: {resolution} {source} {videoCodec} {hdr}
Audio: {audioCodec} {channels}
Other: {group}  |  Pad numbers: {season:2} → 02

Preview:

Audiobook Format

Book: {author} {title} {year} {narrator} {genre}
Series: {series} {bookNum}
Chapter: {chapter} {chapterTitle}
Use / to create folders  |  Pad: {chapter:2} → 01
Empty {series} folders are auto-removed for non-series books.

Preview:

ROM Format

Game: {title} {year} {genre} {developer}
Platform: {platform} {platformShort} {esdeSystem}
File: {region}  |  Use / to create folders
DLC/Update: {contentType} {version}  |  Multi-disc: {disc}
{esdeSystem} = ES-DE folder name (e.g. snes, gba, n3ds). Enable "Use ES-DE names" in Settings to auto-apply.
{contentType} is empty for normal games — path collapses automatically (e.g. Switch/Sonic, not Switch//Sonic).
Right-click any ROM row to manually set disc number or content type.

Preview:

ROM DLC Format

Used automatically when a file is tagged as DLC. Same tokens as ROM Format above.
{contentType} always outputs "DLC" here — or just hardcode the folder name.

Preview:

ROM Update Format

Used automatically when a file is tagged as Update. Same tokens as ROM Format above.
{version} = update version (e.g. v1.1.0). {contentType} always outputs "Update" here.

Settings

Configure API keys and preferences

TMDB API Key

Required for movie and TV show metadata. Get a free key at themoviedb.org

OMDb API Key (optional)

Adds OMDb/IMDb as an additional source. Get a free key at omdbapi.com

IGDB Credentials (required for ROM matching)

Free credentials from dev.twitch.tv — register a Twitch app to get a Client ID and Client Secret. Tokens are cached automatically (~60 day expiry).

ROM Options

Additional options for ROM renaming.

When enabled, {platformShort} outputs ES-DE folder names. You can also use {esdeSystem} in your format string directly.

Free Sources

TVmaze — TV show data, no API key needed. Automatically included in search results.
OpenLibrary — Audiobook/book data, no API key needed.
Internet Archive — ROM name lookup, no API key needed. Available in the ROMs match menu.

Default Output Directory

Where renamed files are placed. Leave empty to rename in place.

Article Sorting

Move leading articles (The, A, An) to the end — e.g. "The Shawshank Redemption" → "Shawshank Redemption, The"
